NIGER — FORMAT OFFICIEL ETHNIAFRICA

1. NOM DU PAYS
Niger (République du Niger)

2. ANCIENNES APPELLATIONS ET HISTOIRE DES NOMS
- Avant la colonisation : pas d’État unifié ; zones touareg, haoussa, zarma-sonraï, peul, toubou, kanouri.
- Le nom « Niger » vient du fleuve Niger (probable origine : “Ni Gir” / “Négher” = eau, fleuve, ou terme berbère).
- Empires précoloniaux influents :
  • Empire songhaï (Gao) : influence sur l’ouest.
  • Empire Kanem-Bornou : influence sur le sud-est.
  • Sultanats haoussa (Katsina, Gobir, Damagaram).
  • Confédérations touarègues (Kel Air, Kel Ifoghas).
- 1900 : Conquête française.
- 1922 : « Colonie du Niger » (AOF).
- 1960 : Indépendance → République du Niger.

3. RÉSUMÉ HISTORIQUE (PRÉCOLONIAL → COLONIAL → MODERNE)
Le Niger actuel est un assemblage de plusieurs sphères historiques : zarma et songhaï dans l’Ouest, haoussa au Sud, kanouri à l’Est, touareg au Nord, peuls transhumants sur l’ensemble du Sahel. Les confédérations touarègues dominent le Nord (Aïr, Azawagh), tandis que le Bornou influence le Sud-Est (Diffa). Le sultanat de Damagaram (Zinder) était un État haoussa puissant. Le pays devient indépendant en 1960.

4. RÉSUMÉ DÉTAILLÉ DES GROUPES ETHNIQUES
- Haoussa (~55 %) : sud du pays ; langue haoussa ; sociétés marchandes.
- Zarma / Songhaï (~22 %) : ouest (Niamey, Tillabéri) ; héritiers de l’empire songhaï.
- Touareg (~10 %) : nomades du Nord ; confédérations Kel Air, Kel Ifoghas ; langue tamasheq.
- Peul / Fulani (~9 %) : pasteurs transhumants ; langue fulfulde ; islam.
- Kanouri (~4 %) : sud-est ; héritiers de l’empire Kanem-Bornou.
- Toubou (~1 %) : nord-est désertique ; langue toubou (teda-daza).
- Gourmantché (~1 %) : sud-ouest ; peuple transfrontalier (Burkina-Faso).
- Arabes Shuwa (~1 %) : pasteurs arabophones du sud-est.
- Autres (~2 %) : migrants et groupes minoritaires.

6. NOTES IMPORTANTES
- Le Niger est un pays sahélien multi-civilisationnel.
- Les Haoussa dominent démographiquement, mais les Zarma structurent l’État.
- Les Touareg possèdent une forte tradition politique au nord (Kel Air).
- Le Kanem-Bornou a façonné la région de Diffa.
