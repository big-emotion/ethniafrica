# Famille linguistique

## HEADER DÉCOLONIAL (obligatoire)

- Identifiant famille linguistique (FLG_xxxxx) : FLG_KROU
- Nom français : Krou
- Nom anglais / auto-appellation académique : Kru
- Appellation(s) historique(s) :
  « Krou » ; « langues krou » ; parfois associées aux ethnonymes « Kru/ Kroo » utilisés dans les archives coloniales maritimes.
- Origine du terme historique :
  Le terme « Krou » dérive des populations côtières connues sous le nom de *Kru* ou *Kroo*, identifiées par les navigateurs européens (XVIIᵉ–XIXᵉ siècles) comme marins et intermédiaires commerciaux.  
  Les linguistes ont ensuite étendu le terme pour désigner un ensemble beaucoup plus large de langues.
- Pourquoi le terme pose problème :
  - « Kru » a historiquement été un terme colonial vague appliqué aux travailleurs côtiers, souvent sans distinction ethnolinguistique.  
  - Les Européens l’ont utilisé comme catégorie professionnelle, effaçant la diversité des groupes (Bété, Dida, Wè, Grebo…).  
  → Le terme moderne « Krou » est linguistiquement valide, mais son héritage colonial doit être expliqué.
- Auto-appellation (locuteurs / linguistes contemporains) :
  Kru languages / Langues krou
- Usage contemporain :
  Le krou est une grande sous-famille du Niger-Congo regroupant des langues de Côte d’Ivoire et du Libéria, dont le bété, le dida, le wê, le grebo, etc.
- Aire / répartition géographique générale :
  Côte d’Ivoire, Liberia, zones forestières côtières
- Nombre de langues (estimation) : ~30–40
- Nombre total de locuteurs (estimation) : ~3 millions

## MODÈLE STRUCTURÉ AFRIK

# 1. Informations générales
- Nom de la famille : Krou
- Branches internes / sous-groupes :
  - Krou oriental : Bété, Dida  
  - Krou occidental : Grebo, Wè/Krahn  
  - Sous-groupes supplémentaires : Godié, Néyo, Bassa (selon classifications)
- Aire géographique :
  Côte d’Ivoire, Libéria
- Nombre de langues :
  30–40
- Nombre total de locuteurs :
  ~3 millions

# 2. Peuples associés
- Peuple 1 : Bété
- Peuple 2 : Dida
- Peuple 3 : Grebo

# 3. Ethnies associées (liste synthétique)
- Ethnie 1 : Bété  
- Ethnie 2 : Dida  
- Ethnie 3 : Wè / Guéré / Krahn  
(+ renvoi vers fiches ethniques dédiées)

# 4. Caractéristiques linguistiques
- Typologie linguistique :
  Langues tonales, à morphologie relativement analytique ; SVO majoritaire.
- Particularités phonologiques :
  - Consonnes prénasalisées  
  - Systèmes vocaliques complexes  
  - Harmonie vocalique dans certaines langues
- Liens avec les familles voisines :
  Contacts intenses avec les langues mandé (au nord) et les langues kwa (au sud-est de la Côte d’Ivoire).
- Innovations clés :
  - Flexion minimale  
  - Importance des tons dans la grammaire  
  - Structure syllabique riche (CVC, CV, parfois CCV)

# 5. Histoire et origines
- Origine probable :
  Forêts guinéo-congolaises de Côte d’Ivoire et du Libéria.
- Diffusion des langues :
  Migration progressive des groupes krou à travers les forêts ivoiriennes ; interactions historiques avec les mandé du nord.
- Ruptures historiques :
  - Colonisation française (Côte d’Ivoire) et américaine/libérienne (Libéria)  
  - Déplacements massifs pendant les guerres civiles libériennes et ivoiriennes
- Zones de contact et métissages :
  - Région ivoirienne de Gagnoa, Daloa, San-Pédro  
  - Littoral libérien  
  - Zones frontalières avec les mandé méridionaux

# 6. Répartition géographique et démographie
- Nombre total de locuteurs :
  ~3 millions
- Répartition par pays :
  Côte d’Ivoire (Bété, Dida, Wè), Libéria (Grebo, Kru)
- Importance historique :
  Les langues krou ont joué un rôle clé dans les résistances anti-coloniales, les circulations maritimes et la construction des identités ivoiriennes.
