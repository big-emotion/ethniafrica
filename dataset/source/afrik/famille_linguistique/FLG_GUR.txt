# Famille linguistique

## HEADER DÉCOLONIAL (obligatoire)

- Identifiant famille linguistique (FLG_xxxxx) : FLG_GUR
- Nom français : Gur
- Nom anglais / auto-appellation académique : Gur / Voltaic (terme ancien)
- Appellation(s) historique(s) :
  « Voltaïque » (ancien terme colonial) ; « Langues voltaïques » (du fleuve Volta).
- Origine du terme historique :
  Le terme « Voltaic » (français : « voltaïque ») a été créé par des administrateurs et linguistes coloniaux français au milieu du XIXe siècle pour désigner les langues parlées autour des affluents du fleuve Volta (Burkina Faso, Ghana).  
  Sigismund Wilhelm Koelle mentionne pour la première fois ces langues sous l'appellation « North-Eastern High Sudan » dans son ouvrage *Polyglotta Africana* (Koelle 1854), correspondant à ce qui sera plus tard classé comme Gur.  
  Le terme « Gur » lui-même a été adopté par les linguistes du XXe siècle, notamment par John T. Bendor-Samuel dans les années 1970 et Gabriel Manessy dans ses travaux sur les langues Oti-Volta (Manessy 1975, 1979), pour remplacer « Voltaic » dans un souci de classification plus cohérente.
- Pourquoi le terme pose problème :
  - « Voltaïque » est un terme purement géographique imposé par l'administration coloniale, sans consultation des communautés locales. Le terme nomme selon un référent géographique (fleuve Volta) sans tenir compte des auto-appellations endogènes.  
  - **Auteurs coloniaux problématiques** : Les premières classifications (Koelle 1854, Manessy dans les années 1950-1970) imposaient des termes externes. Certains linguistes coloniaux incluaient par exemple les langues Sénoufo dans le groupe Gur, classification aujourd'hui rejetée.  
  - **Auteurs locaux défendant l'africanité** : Adams Bodomo (2017, 2020) propose le terme *Mabia* comme remplacement décolonial pour « Gur » ou « Central Gur », terme construit à partir de lexèmes endogènes *ma-* (« mère ») + *-bia* (« enfant ») présents dans plusieurs langues du groupe. Cette proposition est documentée dans *Handbook of the Mabia Languages of West Africa* (Bodomo, Abubakari & Issah 2020). Des populations comme les Mole-Dagbon se nomment elles-mêmes *Mabia* ou « meta-ethnie Mabia » dans certains travaux récents.  
  - **Études génétiques modernes** : L'étude « Contrasting maternal and paternal histories in the linguistic context of Burkina Faso » (de Filippo et al. 2011) montre que les populations parlant les langues Gur présentent une forte fréquence de l'haplogroupe Y-chromosomique E-U175 (près de 67 % chez les Mossi), distincte des groupes Mandé voisins. Ces données génétiques montrent une cohérence partielle entre frontières linguistiques Gur et structure génétique paternelle, contestant l'idée d'un « bloc homogène » imposé par les classifications coloniales.
  → **Le terme moderne correct est "Gur", avec proposition alternative "Mabia" pour le sous-groupe Central Gur / Oti-Volta.**
- Auto-appellation (locuteurs / linguistes contemporains) :
  - **Terme endonyme "Mabia"** : Utilisé par certains locuteurs comme auto-appellation du groupe linguistique Central Gur (anciennement Gur) dans certains contextes académiques et culturels, proposé comme terme plus légitime par Bodomo.  
  - **Étymologie du terme Mabia** : Composante lexicale *ma-* (« mère, mother ») + *-bia* (« enfant, child ») issue de langues de ce groupe. Cette étymologie est attestée dans plusieurs langues Mabia (Dagbani, Mooré, Mamprusi, etc.).  
  - **Attestations historiques** : Bien que « Mabia » soit un terme académique récent (introduit par Bodomo en 2017, institutionnalisé en 2020), les langues du groupe utilisent des mots similaires pour « mère/enfant » dans plusieurs langues Mabia, ce qui soutient l'unité lexicale sous-jacente.  
  - **Usage contemporain** : Bodomo publie *Handbook of the Mabia Languages of West Africa* (2020) en utilisant explicitement Mabia comme terme remplaçant « Central Gur » dans la classification moderne. Les locuteurs de Dagbani, Kusaal, Wala, Sisaali, etc. utilisent ou acceptent l'appellation Mabia.
- Usage contemporain (définition moderne de la famille) :
  Le Gur regroupe des langues majeures du Burkina Faso, du Ghana, du Togo et du Bénin, notamment le mooré (Mooré), le dagbani (Dagbanli), le dagaare, le gulmancema. Le terme « Mabia » est proposé pour désigner spécifiquement le sous-groupe Central Gur / Oti-Volta (Dagbani, Mooré, Mamprusi, etc.).
- Aire / répartition géographique générale :
  Burkina Faso, Ghana, Côte d'Ivoire nord, Togo nord, Bénin nord, sud du Mali, Niger occidental
- Nombre de langues (estimation) : ~70–80
- Nombre total de locuteurs (estimation) : >20 millions

## MODÈLE STRUCTURÉ AFRIK

# 1. Informations générales
- Nom de la famille : Gur
- Branches internes / sous-groupes :
  - Central Gur / Oti-Volta (Mossi/Mooré, Dagbani, Mamprusi, Dagaare, Kusaal, etc.)
  - Northern Gur
  - Southern Gur
  - Grusi / Gurunsi (Kassena, Nankana, Frafra, etc.)
  - Gurma (Gourmantché, Kabiye, etc.)
- Aire géographique :
  Sahel et savanes d'Afrique de l'Ouest, du sud du Mali au nord du Ghana, du Niger occidental au nord du Bénin
- Nombre de langues :
  70–80
- Nombre total de locuteurs :
  >20 millions

# 2. Peuples associés
- Peuple 1 : Mossi (PPL_MOSSI)
- Peuple 2 : Dagomba (PPL_DAGOMBA)
- Peuple 3 : Gourmantché (PPL_GOURMANTCHE)

# 3. Caractéristiques linguistiques
- Typologie linguistique :
  Langues tonales avec structure syntaxique SVO (Sujet-Verbe-Objet) dominante. Exemples concrets : en Mooré, *N na ya* « Je mange riz » (*N* = je, *na* = mange, *ya* = riz) ; en Dagbani, *Dagbo langa* « langue de Dagbo » montre l'ordre possesseur-possédé avec postpositions.  
  Système de classes nominales marqué par des suffixes dans certaines langues Gur. Exemple : dans les langues Oti-Volta, les noms peuvent être marqués par des suffixes de classe (-li, -la) qui influencent l'accord des adjectifs et déterminants.  
  Distinction aspectuelle perfectif/imperfectif dans les verbes. Exemple : en Gur (Dagbani), les formes verbales marquent l'aspect achevé vs non-achevé via des affixes ou alternances internes (contraste entre passé accompli vs non accompli).
- Particularités phonologiques :
  **Tonalité** : Systèmes tonals avec généralement 2 à 4 tons contrastifs. Les tons peuvent porter des distinctions grammaticales (ex : temps/aspect) ainsi que lexicales. Exemple en Dagaare : *dá* (ton haut) « pousser » vs *dà* (ton bas) « acheter ». Certaines langues comme Koromfé sont atonales (cas rare dans Gur).  
  **Nasales syllabiques** : Usage de nasales comme syllabes à elles seules en position initiale. Exemples : en Dagbani, *nzugu* « ma tête » ; *mbia* « mon enfant » dont la syllabe initiale est nasale et porte ton.  
  **Consonnes** : Inventaire consonantique riche comprenant stops (voisés/non-voisés), fricatives, nasales, labi-vélaires (kp, gb), et parfois implosives (/ɓ/, /ɗ/) dans certaines langues comme Dagaare.  
  **Harmonie vocalique** : Dans certaines langues de la branche Grusi (ex : Dagaare, Waale), harmonie basée sur ATR (Advanced/Non-Advanced Tongue Root) avec longueur vocalique distinctive.
- Liens avec les familles voisines :
  Emprunts lexicaux intenses avec les langues Mandé (Bambara, Malinké) dans les zones de contact, notamment dans le vocabulaire du commerce et de la végétation.  
  Contacts avec les langues Kwa : les langues Gurunsi orientales (selon Kleinewillinghöfer 2002) montrent des influences des langues Kwa avec des emprunts lexicaux spécifiques.  
  Emprunts entre langues Gur voisines : exemple, mots partagés comme *di* « manger » (Mooré *di*, Dagaare *di*) et *wɛ/kõwõ* « eau » (Mooré *wɛ*, Dagaare *kõwõ*) montrent des lexicalismes communs au sein du groupe.
- Innovations clés :
  - Développement des systèmes tonals grammaticaux utilisant les tons pour marquer des catégories grammaticales (nombre, aspect, classes nominales) en plus des distinctions lexicales
  - Simplification ou absence de certaines classes nominales archaïques dans certains Gur modernes (ex : certaines variétés de Safaliba où les suffixes de classes deviennent neutres)
  - Riches alternances vocaliques avec harmonie ATR dans certaines langues
  - Innovation lexicale *ma-/bia* partagée dans le groupe Mabia (Central Gur), non présente dans les langues Mandé voisines

# 4. Histoire et origines
- Origine probable (théories archéologiques, linguistiques, génétiques) :
  **Théories linguistiques** : Sigismund Wilhelm Koelle (1854) mentionne pour la première fois des langues que l'on classerait aujourd'hui dans Gur sous l'appellation « North-Eastern High Sudan » dans *Polyglotta Africana*. Joseph H. Greenberg (1955, révisé 1963) dans *The Languages of Africa* classifie les langues alors appelées Voltaic comme branche du Niger-Congo. Gabriel Manessy (1975, 1979) contribue à la classification généalogique des langues voltaïques avec ses travaux sur le proto-central et les langues Oti-Volta. Tony Naden (1989) et Williamson & Blench (2000) affinent les subdivisions et rejettent l'inclusion des Sénoufo dans Gur. Ulrich Kleinewillinghöfer (2014) travaille sur les connexions Gur-Adamawa et les analyses typologiques modernes.  
  **Théories archéologiques** : Les données archéologiques suggèrent une diffusion des pratiques agricoles (millet, sorgho) dans les savanes ouest-africaines, probablement il y a quelques millénaires (estimations variant entre ~3000-2000 av. J.-C. selon certains modèles), mais les preuves précises restent indirectes.  
  **Théories génétiques** : L'étude « Contrasting maternal and paternal histories in the linguistic context of Burkina Faso » (de Filippo et al. 2011) montre que les populations parlant les langues Gur partagent des traits génétiques communs, notamment une forte fréquence de l'haplogroupe Y-chromosomique E-U175 (près de 67 % chez les Mossi), distincts des groupes Mandé voisins, indiquant une histoire de peuplement locale cohérente avec les frontières linguistiques.
- Période d'émergence estimée :
  **1. Émergence historique de la FLG** : Reconstruction linguistique suggère que le proto-Gur/Central Gur était déjà distinct il y a plusieurs millénaires, probablement au cours du 2e millénaire avant notre ère selon les modèles basés sur les technologies agricoles et migrations dans les savanes ouest-africaines. Certaines estimations archéologiques placent cette émergence dans le Néolithique ou Bronze ancien (~3000-2000 av. J.-C.), mais les données précises manquent et restent indirectes.  
  **2. Date de classification académique** : Le terme « Voltaic » apparaît au milieu du XIXe siècle (Koelle 1854). Joseph Greenberg (1955, révisé 1963) classe les langues Voltaic comme branche du Niger-Congo. Le terme « Gur » est adopté dans les années 1970 par Bendor-Samuel et Manessy pour remplacer « Voltaic ». Le terme « Mabia » est proposé par Bodomo en 2017 et institutionnalisé en 2020 avec la publication du *Handbook of the Mabia Languages of West Africa*.
- Diffusion des langues (routes migratoires, expansion) :
  **Théorie 1** : Expansion depuis le bassin de la Volta (région actuelle Burkina Faso-nord Ghana-Togo) par migration graduelle vers le nord et l'est, avec sédentarisation dans la savane boisée. Expansion des royaumes mossi (XVe–XIXe siècles) comme facteur de diffusion, migrations agricoles dans la zone voltaïque.  
  **Théorie 2** : Gradient de contact linguistique avec les langues Mande, Gurunsi, Kwa influençant la diffusion culturelle et lexicale. Kleinewillinghöfer (2014) propose que beaucoup de langues considérées dans Adamawa pourraient être plus proches de Gur que d'autres Adamawa, suggérant un continuum linguistique (savannas languages) plutôt qu'une séparation nette.  
  **Théorie 3** : Influence environnementale (climatique, écologique) sur la diffusion, avec adaptation agriculture ↔ langues. Certaines études génétiques suggèrent des expansions démographiques liées au climat (Holocene Climate Optimum) favorisant la dispersion nord-sud-est des locuteurs de proto-Gur/proto-Mabia.
- Ruptures historiques (séparations, fusions) :
  - Pressions mandingues au Moyen Âge affectant les zones frontalières
  - Colonisation française et britannique avec imposition de frontières administratives
  - Rejet académique de l'inclusion des Sénoufo dans Gur (Naden 1989, Williamson & Blench 2000)
  - Déplacements liés à la désertification et aux conflits sahéliens récents
- Zones de contact et métissages :
  **Géographie précise** : Frontière Burkina Faso / Côte d'Ivoire pour les langues Sénoufo ; zones nord Ghana / sud Burkina pour langues Oti-Volta ; zones nord Togo / nord Bénin avec Grusi ; régions sahéliennes et de savane au nord de la forêt de Côte d'Ivoire, du Mali central, Burkina Faso, nord Ghana, nord Togo, nord Bénin, sud-désert du Niger.  
  **Villes/routes précises** : Corridors de commerce entre Ouagadougou (Burkina Faso), Kumasi (Ghana) via zones frontalières ; marchés transfrontaliers Burkina-Ghana ; routes commerciales précoloniales du Sahel-Savane (trans-sahariennes, échanges avec Mossi, Gonja) facilitant emprunts et échanges linguistiques.  
  **Zones de métissage linguistique spécifiques** : Langues Gurunsi en zones de contact Kwa (régions orientales de Ghana/Togo) selon Kleinewillinghöfer (2002) ; zones côtières nord-Côte d'Ivoire, nord-Bénin ; zones autour de la Volta blanche/Volta noire dans Ghana.
- Événements historiques majeurs ayant influencé la famille :
  - **~1854** : Sigismund Wilhelm Koelle publie *Polyglotta Africana* mentionnant des langues « North-Eastern High Sudan » (futur Gur)
  - **1955/1963** : Joseph Greenberg publie *The Languages of Africa* (1955, révisé 1963) classifiant les langues Gur/Voltaic comme branche du Niger-Congo
  - **Années 1970** : Adoption du terme « Gur » par Bendor-Samuel et Manessy pour remplacer « Voltaic » ; travaux de Manessy (1975, 1979) sur les langues Oti-Volta et le proto-central
  - **1989** : Tony Naden publie travaux linguistiques sur Gur/Oti-Volta et rejette l'inclusion des Sénoufo dans Gur
  - **2000** : Williamson & Blench publient *Niger-Congo Linguistic Classification* confirmant le rejet des Sénoufo dans Gur
  - **2017** : Adams Bodomo présente « Mabia » comme terme alternatif décolonial pour Central Gur
  - **2020** : Publication de *Handbook of the Mabia Languages of West Africa* (Bodomo, Abubakari & Issah 2020) institutionnalisant le terme Mabia

# 5. Répartition géographique et démographie
- Nombre total de locuteurs :
  >20 millions (estimations 2025)
- Répartition par pays :
  **Burkina Faso** : ~10-12 millions de locuteurs Gur (estimation ~35-40 % de la population nationale). Langue nationale : mooré (Mooré). Autres langues : Gourmantché (~6,8 %), Bissa (~3,3 %), Gurunsi (~3,2 %). Sources : CIA World Factbook, recensements nationaux 2022-2024.  
  **Ghana** : ~5-7 millions de locuteurs Gur (estimation ~15-20 % de la population nationale). Langues principales : Dagbani (Dagbanli), Mamprusi (Mampruli), Dagaare, Kusaal, Frafra, etc., surtout dans les régions nord (Upper East, Upper West, Northern Region). Sources : Britannica, Ethnologue, Bodomo 2020.  
  **Togo** : ~1-2 millions de locuteurs Gur (estimation ~10-15 % selon régions). Langues principales : Kabiye (~550 000 L1, ~1,2 million avec L2), Gourmanchéma, etc., surtout dans le nord. Sources : Britannica, SIL.  
  **Bénin** : ~1 million de locuteurs Gur (estimation ~5-10 %), surtout dans le nord-ouest. Langues Oti-Volta et Gurma. Sources : Britannica, LanguagesGulper.  
  **Côte d'Ivoire** : ~2 millions de locuteurs Gur dans le nord (estimation ~5-10 %). Langues Sénoufo, Grusi, Oti-Volta. Sources : Britannica, SIL.  
  **Mali, Niger** : Petites populations Gur (dans le sud de ces pays), quelques centaines de milliers (~1-3 % selon région).  
  Sources précises : CIA World Factbook (2024-2025), SIL Ethnologue (2024-2025), Britannica (2024-2025), recensements nationaux ONU/UNFPA (2022-2025), Bodomo 2020, LanguagesGulper.
