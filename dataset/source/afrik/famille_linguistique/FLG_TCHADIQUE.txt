# Famille linguistique

## HEADER DÉCOLONIAL (obligatoire)

- Identifiant famille linguistique (FLG_xxxxx) : FLG_TCHADIQUE
- Nom français : Tchadique
- Nom anglais / auto-appellation académique : Chadic
- Appellation(s) historique(s) :
  « Hamitique » (désignation raciale coloniale), « Langues tchadiennes », « Sous-groupe tchadique ».
- Origine du terme historique :
  Le terme « tchadique » vient du bassin du lac Tchad, où se trouvent les principales langues du groupe.  
  Dans la classification du XIXᵉ siècle, ces langues étaient incluses dans l’ensemble « hamitique », hérité d’une lecture biblique et eurocentrée.
- Pourquoi le terme pose problème :
  - L’étiquette « hamite » associait à tort des peuples africains à une origine non-africaine prétendument « civilisatrice ».  
  - Le regroupement « hamitique » servait une hiérarchie raciale artificielle.  
  - Aujourd’hui, l’appellation **tchadique** (linguistique) est scientifiquement valide mais son historique colonial exige contextualisation.
- Auto-appellation (locuteurs / linguistes contemporains) :
  Chadic languages / Langues tchadiques
- Usage contemporain :
  Branche importante de l’Afro-asiatique, comprenant plus de 150 langues parlées autour du lac Tchad, dominée démographiquement par le haoussa.
- Aire / répartition géographique générale :
  Nigeria, Niger, Tchad, Cameroun, nord de la RCA.
- Nombre de langues (estimation) : 150–190
- Nombre total de locuteurs (estimation) : >70 millions

# 1. Informations générales
- Nom de la famille : Tchadique

# 4. Caractéristiques linguistiques
- Typologie linguistique :
  Langues tonales, structures verbales complexes, flexion nominale relativement simple.
- Particularités phonologiques :
  Consonnes implosives ; forte tonalité ; consonnes labio-vélaires dans certaines langues.
- Liens avec les familles voisines :
  Influence réciproque avec les langues nilo-sahariennes (Saharien, Kanuri notamment) et les langues nigéro-congolaises voisines.
- Innovations clés :
  - Développement ancien de structures tonales complexes  
  - Importance sociolinguistique du haoussa comme lingua franca  
  - Systèmes verbaux dérivationnels multiples

# 5. Histoire et origines
- Origine probable :
  Zones sahéliennes proches du lac Tchad, au nord du Nigeria actuel.
- Diffusion des langues :
  Expansion du haoussa par le commerce transsaharien ; diffusion dans les routes du Sahel ouest.
- Ruptures historiques :
  - Déplacement de populations (Kanuri vs Hausa)  
  - Islamisation des cités-États haoussa  
  - Pressions coloniales françaises et britanniques
- Zones de contact et métissages :
  - Contacts anciens avec les Songhay  
  - Métissage linguistique avec le Kanuri  
  - Interactions avec les peuples bantous du Cameroun (Biu-Mandara)

# 6. Répartition géographique et démographie
- Nombre total de locuteurs :
  >70 millions
- Répartition par pays :
  Nigeria (forte majorité), Niger, Tchad, Cameroun, nord de la RCA
- Importance historique :
  Langue véhiculaire essentielle du Sahel occidental ; cités-États haoussa ; influence massive dans le commerce transsaharien.
