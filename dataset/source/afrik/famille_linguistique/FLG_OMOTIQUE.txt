# Famille linguistique

## HEADER DÉCOLONIAL (obligatoire)

- Identifiant famille linguistique (FLG_xxxxx) : FLG_OMOTIQUE
- Nom français : Omotique
- Nom anglais / auto-appellation académique : Omotic
- Appellation(s) historique(s) :
  Anciennement considérée comme une sous-branche du couchitique (« Western Cushitic »).
- Origine du terme historique :
  Le nom « Omotic » vient de la rivière Omo en Éthiopie, région où sont parlées ces langues.  
  La classification omotique comme branche distincte a émergé dans la deuxième moitié du XXᵉ siècle.
- Pourquoi le terme pose problème :
  - Le rattachement ancien au « couchitique » faisait partie d’une classification hamitique dépassée.  
  - Le statut de l’omotique reste controversé : certaines études suggèrent qu’il pourrait être une famille indépendante, ou un lien plus ancien au sein de l’afro-asiatique.
  → Le problème n’est pas colonial comme dans d’autres familles, mais **scientifique** : l’omotique reste l’une des branches les plus débattues.
- Auto-appellation (locuteurs / linguistes contemporains) :
  Omotic languages / Langues omotiques
- Usage contemporain :
  Ensemble de langues situées principalement au sud-ouest de l’Éthiopie, parlées par plusieurs dizaines de groupes agricoles dans la vallée de l’Omo.
- Aire / répartition géographique générale :
  Sud-ouest de l’Éthiopie (vallée de l’Omo, zones montagneuses voisines)
- Nombre de langues (estimation) : ~25–30
- Nombre total de locuteurs (estimation) : ~4 à 5 millions

# 4. Caractéristiques linguistiques
- Typologie linguistique :
  Langues tonales ou accentuelles, fortement suffixantes ; structure SOV fréquente.
- Particularités phonologiques :
  Systèmes consonantiques variés ; harmonies vocaliques dans certaines langues ; distinctions de longueur vocalique.
- Liens avec les familles voisines :
  Contacts anciens avec les langues couchitiques et sémitiques éthiopiennes.
- Innovations clés :
  - Flexion verbale complexe  
  - Suffixation multiple  
  - Usage étendu de particules aspectuelles

# 5. Histoire et origines
- Origine probable :
  Région des hautes terres et basses terres de l’Omo.
- Diffusion des langues :
  Répartition stable depuis plusieurs siècles dans le sud-ouest éthiopien.
- Ruptures historiques :
  - Domination du royaume éthiopien (XIXe–XXe siècles)
  - Pression démographique et linguistique des langues couchitiques et sémitiques
- Zones de contact et métissages :
  - Forts échanges avec les peuples couchitiques  
  - Influence oromo dans certaines zones  
  - Contacts interethniques via marchés régionaux

# 6. Répartition géographique et démographie
- Nombre total de locuteurs :
  ~4 à 5 millions
- Répartition par pays :
  Éthiopie uniquement (principalement régions de Wolaytta, Gamo, Gofa, Ari)
- Importance historique :
  Région linguistiquement conservatrice de l’Éthiopie ; rôle clé dans la compréhension de l’histoire afro-asiatique.
