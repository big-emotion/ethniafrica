# Famille linguistique

## HEADER DÉCOLONIAL (obligatoire)

- Identifiant famille linguistique (FLG_xxxxx) : FLG_MANDE
- Nom français : Mandé
- Nom anglais / auto-appellation académique : Mande
- Appellation(s) historique(s) :
  « Mandingue » (souvent utilisé de manière trop large),  
  « Langues mandé »,  
  parfois appelées « niger-congolaises atypiques » dans la littérature linguistique.
- Origine du terme historique :
  « Mandé » vient des peuples et langues du Mandé historique (Mali, Guinée, Côte d’Ivoire).  
  Les Français utilisaient « mandingue » pour désigner plusieurs populations distinctes (Bambara, Malinké, Dioula), simplifiant des identités complexes.
- Pourquoi le terme pose problème :
  - « Mandingue » est un exonyme colonial trop large, qui engloutit plusieurs ethnies (Bambara, Malinké, Dioula, Maninka) sous un même nom.  
  - Il véhicule l’idée fausse d’un groupe homogène.  
  - Le terme correct est **Mandé**, qui désigne l’ensemble linguistique, et non un unique peuple.
- Auto-appellation (locuteurs / linguistes contemporains) :
  Mande languages / Langues mandé
- Usage contemporain :
  Le mandé est une branche du Niger-Congo caractérisée par l’absence de classes nominales et un système verbal complexe.  
  Il inclut le bambara, le malinké, le soninké, le susu, le kpelle, etc.
- Aire / répartition géographique générale :
  Mali, Guinée, Côte d’Ivoire, Burkina Faso, Sierra Leone, Sénégal, Gambie, Liberia
- Nombre de langues (estimation) : ~60–70
- Nombre total de locuteurs (estimation) : ~30 millions

## MODÈLE STRUCTURÉ AFRIK

# 1. Informations générales
- Nom de la famille : Mandé
- Branches internes / sous-groupes :
  - Mandé occidental : Bambara, Malinké, Mandinka, Dioula, Soninké, Susu  
  - Mandé oriental : Bissa, Boko, Kyenga  
  - Sous-groupes sud : Kpelle, Dan, Mano
- Aire géographique :
  Mali, Guinée, Côte d’Ivoire, Burkina Faso, Sierra Leone, Liberia
- Nombre de langues :
  60–70
- Nombre total de locuteurs :
  ~30 millions

# 2. Peuples associés
- Peuple 1 : Mandingues (Bambara, Malinké, Mandinka, Dioula)
- Peuple 2 : Soninké
- Peuple 3 : Kpelle

# 3. Ethnies associées (liste synthétique)
- Ethnie 1 : Bambara
- Ethnie 2 : Malinké
- Ethnie 3 : Soninké
(+ renvoi aux futures fiches ethniques)

# 4. Caractéristiques linguistiques
- Typologie linguistique :
  Langues majoritairement tonales, SOV, morphologie analytique avec particules aspectuelles.
- Particularités phonologiques :
  Consonnes prénasalisées, tons lexicaux, harmoniques vocaliques dans certaines branches.
- Liens avec les familles voisines :
  Forte interaction historique avec les langues atlantiques (wolof, peul) et les langues gur.
- Innovations clés :
  - Absence ou réduction des classes nominales (différence majeure avec le reste du Niger-Congo)  
  - Systèmes verbaux aspectuels riches  
  - Système tonal complexe

# 5. Histoire et origines
- Origine probable :
  Région des sources du Niger, entre Guinée et Mali actuels.
- Diffusion des langues :
  Expansion mandingue médiévale (Mali, Kaabu, empire mandingue), migrations économiques modernes (Dioula).
- Ruptures historiques :
  - Chute de l’empire du Mali  
  - Islamisation successive des peuples mandé  
  - Pressions coloniales françaises (imposition de nouvelles frontières)
- Zones de contact et métissages :
  Pays mandingues du Mali/Guinée, Sénégambie, Sierra Leone.

# 6. Répartition géographique et démographie
- Nombre total de locuteurs :
  ~30 millions
- Répartition par pays :
  Guinée, Mali, Côte d’Ivoire, Burkina Faso, Gambie, Sénégal, Sierra Leone, Liberia
- Importance historique :
  Langues de l’empire du Mali, de Sundiata Keïta, et des grands réseaux transsahariens.
