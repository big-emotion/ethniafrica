TCHAD — FORMAT OFFICIEL ETHNIAFRICA

1. NOM DU PAYS
Tchad (République du Tchad)

2. ANCIENNES APPELLATIONS ET HISTOIRE DES NOMS
- Avant la colonisation : pas d’État unifié ; empires sahéliens et royaumes soudanais.
- Grands ensembles historiques :
  • Empire du Kanem (VIIIe–XIVe siècle) : autour du lac Tchad.
  • Empire du Bornou (XIVe–XIXe siècle) : domination kanouri.
  • Royaume du Baguirmi (XVIe–XIXe siècle) : peuple Barma.
  • Royaume du Ouaddaï (XVIe–XIXe siècle) : dynastie arabe mahri.
- Nom “Tchad” : vient du terme local “Tsade / Chad” signifiant “grand lac”.
- 1900–1920 : conquête française.
- 1920 : Colonie du Tchad (AEF).
- 1960 : Indépendance → République du Tchad.

3. RÉSUMÉ HISTORIQUE (PRÉCOLONIAL → COLONIAL → MODERNE)
Le Tchad réunit trois zones :
- Nord saharien (Touareg, Toubou).
- Centre sahélien (Arabes, Zaghawa, Bideyat).
- Sud soudanais (Sara, Ngambaye, Mbaye, Goula…).
Les empires Kanem-Bornou, Baguirmi et Ouaddaï structurent la région pendant plus d’un millénaire. La France intègre le pays dans l’AEF avant l’indépendance en 1960.

4. RÉSUMÉ DÉTAILLÉ DES GROUPES ETHNIQUES
- Sara (~28 %) : sud ; peuples Ngambaye, Mbaye, Kaba ; agriculteurs sédentaires.
- Arabes tchadiens (~12 %) : nomades et semi-nomades ; langue arabe tchadien ; islam.
- Kanembou (~9 %) : peuple du lac Tchad ; héritiers du Kanem-Bornou.
- Hadjarai (~7 %) : groupes montagnards du centre ; langues tchadiques.
- Gorane / Toubou (~7 %) : nord-est ; nomades du Sahara.
- Massa / Musey (~5 %) : sud-ouest ; langue massa.
- Baguirmi (~4 %) : héritiers du royaume Barma.
- Fulani / Peul (~3 %) : pasteurs transhumants.
- Zaghawa / Bideyat (~4 %) : nord-est ; peuple saharien lié au Soudan.
- Moundang (~3 %) : sud-ouest ; langue moundang.
- Tupuri (~2 %) : peuple sahélien du sud-est.
- Ouaddaïens / Mahamid (~2 %) : est ; héritiers du Ouaddaï.
- Autres groupes (~14 %) : Goula, Ngama, Ndam, Kaba, Marba, Barma, etc.

6. NOTES IMPORTANTES
- Le Tchad est un pays-carrefour entre Afrique noire et Sahara.
- Le Sud (Sara) structure l’économie agricole ; le Nord est saharien et nomade.
- Le Kanem-Bornou est l’un des plus anciens empires sahéliens.
- Les Zaghawa sont transfrontaliers avec le Soudan (Darfour).
