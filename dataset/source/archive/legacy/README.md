# Données legacy

Ce dossier contient les anciennes données organisées par région qui ont été remplacées par la structure AFRIK.

## Structure

- `afrique_australe/` - Données de l'Afrique australe (format ancien)
- `afrique_centrale/` - Données de l'Afrique centrale (format ancien)
- `afrique_de_l_est/` - Données de l'Afrique de l'Est (format ancien)
- `afrique_de_l_ouest/` - Données de l'Afrique de l'Ouest (format ancien)
- `afrique_du_nord/` - Données de l'Afrique du Nord (format ancien)
- `matched/` - Fichiers JSON de matching (ancien format)
- `parsed/` - Fichiers JSON parsés (ancien format)

## Note

Ces données ont été remplacées par la nouvelle structure AFRIK organisée par :

- Familles linguistiques (`/dataset/source/afrik/famille_linguistique/`)
- Peuples organisés par famille linguistique (`/dataset/source/afrik/peuples/FLG_*/`)
- Pays (`/dataset/source/afrik/pays/`)

Les données legacy sont conservées pour référence historique mais ne sont plus utilisées dans le workflow actuel.
