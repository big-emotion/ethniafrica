# Authorized sources

Only sources from this list count as **primary** citations. Other sources (Wikipedia, news outlets, blogs, social media) may be used as pointers / leads but must be replaced by a primary citation before the proposal is finalized.

## Academic / archival

- Peer-reviewed journals (Africa, JAH, IJAHS, Cahiers d'études africaines) — primary if cited with DOI / persistent URL.
- University presses (CUP, OUP, UC Press, Karthala, Présence Africaine).
- National archives (e.g. Archives nationales du Sénégal, SOAS).

When citing academic sources, use the format:

```
"Title – Author(s), Year (DOI or URL if available)"
```

## Pointers (NOT primary)

These can lead you to primary sources but cannot stand alone:

- Wikipedia (use to find references at the bottom of articles, then cite those directly)
- News outlets (RFI, BBC Afrique, Jeune Afrique) — usable for current events only
- Cultural / community sites — flag in `appellationRemarks` if useful but not primary

## Rejection criteria

Reject as primary source: blog posts, social media, AI-generated content, undated material, sources without an identifiable author / institution.

When no primary source is available for a field, the field stays `null` and the gap is listed in "Open questions / data still missing".
