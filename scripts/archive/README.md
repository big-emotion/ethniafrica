# Scripts archivés

Ce dossier contient les scripts liés aux étapes passées du projet qui ne sont plus utilisés dans le workflow actuel.

## Scripts d'enrichissement (Étape 4)

- `enrichEthniesAutomated.ts` - Enrichissement automatisé des ethnies
- `enrichEthniesBatch.ts` - Enrichissement par lots
- `enrichEthniesWithBrowserbase.ts` - Enrichissement via Browserbase
- `selectEthniesToEnrich.ts` - Sélection des ethnies à enrichir
- `generateEnrichmentReport.ts` - Génération de rapports d'enrichissement

## Scripts de génération d'ethnies

- `generateEthnieFiles.ts` - Génération de fichiers ethnies
- `updateEthnieFiles.ts` - Mise à jour des fichiers ethnies
- `consolidateEthnieData.ts` - Consolidation des données ethnies

## Scripts de sous-groupes (Étape 5)

- `analyzeSubgroupsETAPE5.ts` - Analyse des sous-groupes
- `generateSubgroupsETAPE5.ts` - Génération des sous-groupes
- `updateEthniesWithSubgroups.ts` - Mise à jour avec sous-groupes
- `checkEthniesAndUpdateWorkflow.ts` - Vérification et mise à jour du workflow

## Scripts de nettoyage

- `clean_rural_urban_diaspora.py` - Nettoyage des variantes rural/urbain/diaspora

## Note

Ces scripts sont conservés pour référence historique mais ne font plus partie du workflow actuel du projet AFRIK, qui se concentre désormais sur les familles linguistiques, langues, peuples et pays.
