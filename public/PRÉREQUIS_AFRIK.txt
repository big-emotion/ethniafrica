# PRÉREQUIS_OFFICIELS_DU_PROJET_AFRIK
# VERSION 2.1 — Document maître (Architecture + Workflow + Identifiants)


========================================================
0. OBJECTIF DU DOCUMENT
========================================================

Ce document établit :

• l’architecture du projet Afrik  
• les identifiants uniques  
• le référentiel linguistique  
• la classification famille → peuple → ethnie → sous-ethnie → clan  
• les sources officielles  
• la méthodologie démographique  
• les règles de nommage  
• les fichiers maîtres CSV  
• le workflow global  
• les fichiers de suivi du workflow


========================================================
1. IDENTIFIANTS UNIQUES AFRIK
========================================================

ID pays : ISO 3166-1 alpha-3  
ID famille linguistique : FLG_xxxx  
ID langue : code ISO 639-3  
ID peuple : PPL_xxxx  
ID ethnie : ETH_xxxx  
ID sous-ethnie : SUB_xxxx  
ID clan : CLN_xxxx  


========================================================
2. RÉFÉRENTIEL LINGUISTIQUE AFRIK
========================================================

Références obligatoires :

• ISO 639-3  
• Glottolog  
• Ethnologue (SIL)  
• UNESCO  
• African Language Atlas  

Chaque langue doit indiquer :

• nom  
• code ISO  
• famille linguistique  
• sous-famille  
• nombre de locuteurs  


========================================================
3. HIÉRARCHIE AFRIK
========================================================

Niveau 1 — Famille linguistique  
Niveau 2 — Peuple  
Niveau 3 — Ethnie  
Niveau 4 — Sous-ethnie  
Niveau 5 — Clan  


========================================================
4. TERMES PÉJORATIFS / COLONIAUX
========================================================

Obligatoire :

• conserver  
• contextualiser  
• expliquer origine  
• expliquer problème  
• donner auto-appellation  
• ne jamais présenter comme neutre  


========================================================
5. MÉTHODOLOGIE DÉMOGRAPHIQUE
========================================================

Population pays = ONU / UNFPA (2025)  
% ethniques = CIA / SIL / UNESCO + anthropologie  
Somme = 100 %  
Chaque peuple = somme ethnies  
Chaque ethnie = somme par pays  
Langue = total locuteurs

========================================================

========================================================

========================================================
