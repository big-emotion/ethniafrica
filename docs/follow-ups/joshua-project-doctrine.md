# Follow-up: Joshua Project — doctrine call needed

Surfaced during the Wikipedia sweep for issue #106. Joshua Project entries
appear in approximately **507 active AFRIK fiches** (peoples, language
families, countries) in their `sources` arrays — far more than the rough
"6×" estimate from the original ticket. The Source Tier Policy in
`CLAUDE.md` enumerates Tier 1 (UN, UNFPA, CIA, SIL Ethnologue, Glottolog,
UNESCO, IWGIA) and does not mention Joshua Project; it is therefore
currently ambiguous whether Joshua Project counts as a Tier-1 authorized
source, a Tier-2 fallback (it is itself a secondary aggregator), or a
Tier-3 forbidden citation (evangelical-mission origin, methodology not
peer-reviewed). A formal doctrine ruling is needed before any sweep can
mechanically remove or downgrade these citations.
