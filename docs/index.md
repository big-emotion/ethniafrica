## Domain Quick-Ref (AFRIK)

Hierarchy: **Linguistic Family → Language → People → Country**

- Families: `FLG_xxxxx` (e.g. `FLG_BANTU`)
- Languages: ISO 639-3 (e.g. `swa`)
- Peoples: `PPL_xxxxx` (e.g. `PPL_YORUBA`)
- Countries: ISO 3166-1 alpha-3 (e.g. `ZAF`)

Details: [Data Models](./data-models.md).

## Before You Code

Read **`_bmad-output/project-context.md`** — it contains the non-obvious rules agents must follow:

- Supabase client isolation (browser / server / admin)
- AFRIK data discipline (strict models, authorized sources, 2025 demographics sum to 100%)
- French-only UI (`Language = "fr"`; multi-lang shape preserved)
- V1 → V2 migration complete — do not reintroduce V1 entities
- Storybook: `@storybook/react-vite` only (not `@storybook/nextjs`)
- Mobile-first design (review at 320–430px first)
- No `Co-Authored-By` trailers
- Code-English / UI-French / Chat-French split
